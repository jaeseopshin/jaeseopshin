## 🚀 Key Projects

### 1. 엔터프라이즈 이기종 라우팅 인프라 & Cisco ASA 보안 구축
* **설명:** OSPF, EIGRP, RIPv2 등 이기종 라우팅 프로토콜 환경을 연동하고 방화벽 보안구역을 분할한 대규모 엔터프라이즈 네트워크 인프라 프로젝트
* **주요 역할 (조장 및 라우팅 실무 전담):**
  * VLAN, VTP, MSTP 설정을 통한 스위칭 루트 이중화 및 대역폭 확장(EtherChannel) 구성
  * 프로토콜 간 상호 재분배(Redistribution) 통신망 구축 및 MD5 라우팅 인증을 통한 원격 제어 보안 설정
  * Cisco ASA 방화벽 가상화 및 가상 컨텍스트(Multiple Context Mode) 생성을 통한 DMZ 복합 NAT 정책 수립
  * 네트워크 전 구간 설정값 검증, 트래픽 경로 분석 및 프로젝트 최종 기술서 작성 주도

### 2. 테라폼(IaC) 기반 AWS 클라우드 고가용성 3-Tier 아키텍처 자동화
* **설명:** AWS 환경에서 고가용성과 보안성을 확보한 격리된 가상 네트워크를 설계하고, 테라폼 코드로 전체 인프라 배포를 자동화한 프로젝트
* **주요 역할 (프로젝트 총괄 및 IaC 설계):**
  * 가용 영역(AZ) 분산 배치를 고려한 격리된 VPC 및 퍼블릭/프라이빗 서브넷 이중화 네트워크 설계
  * Application Load Balancer(ALB) 및 NAT 게이트웨이 연동을 통한 네트워크 부하 분산 최적화
  * 리소스 자원 사양 및 상세 설정값을 체계적으로 문서화한 '인프라 관리대장' 총괄 작성
  * VPC, RDS, ALB 등의 리소스를 테라폼 코드(HCL)로 구조화하여 GitHub-Terraform Cloud 연동 배포 자동화 구현

### 3. Kubernetes 클러스터 운영 관리 및 ECS 서버리스 오케스트레이션
* **설명:** 컨테이너 환경의 가용성 확보 및 보안 정책 검증을 위해 클러스터 레벨의 권한 제어와 서버리스 컨테이너 인프라를 검증한 프로젝트
* **주요 역할 (조장 및 클러스터 실무 전담):**
  * ETCD 백업/복원, Kubeadm 클러스터 업그레이드 및 서비스 어카운트 기반 RBAC 보안 설정 주도
  * 인그레스(Ingress) 및 네트워크 정책 수립을 통한 가용성 워크로드 관리, PV/PVC 영구 스토리지 데이터 공유 운영
  * AWS Fargate 기반 가상 클러스터 구축, Nginx 도커 이미지 빌드 후 프라이빗 ECR 저장소 이중 푸시 관리 수행
  * DNS 룩업 및 파드 로그 검증 테스트 수행 결과를 바탕으로 최종 기술 보고서 작성 주도
