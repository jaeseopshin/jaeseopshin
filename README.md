* **버전/운용도구:** Rocky Linux 9

* **나의 역할:** 프로젝트 총괄 및 인프라 서버 구축 실무 전담, 전 과정 서버 세팅 및 기술 문서 작성용 데이터 캡처/지식 자산화 주도

### 2. 이기종 라우팅 프로토콜 환경 엔터프라이즈 네트워크 인프라 구축
* **버전/운용도구:** Web-IOU / Xshell
* **상세 내용:** 
  * VLAN, VTP, MSTP 설정을 통한 스위칭 루트 이중화 및 EtherChannel 대역폭 확장 구성
  * OSPF, EIGRP, RIPv2 프로토콜 연동 및 상호 재분배(Redistribution) 통신망 구축
  * MD5 라우팅 인증 적용, SSH 원격 제어 보안 설정 및 FTP 기반 장애 덤프 전송 체계 수립
* **나의 역할:** 조장으로서 네트워크 토폴로지 설계 및 OSPF/EIGRP 상호 재분배 라우팅 실무 전담, 네트워크 전 구간 설정값 검증, 트래픽 경로 분석 및 프로젝트 기술서 최종 작성

### 3. Cisco ASA 방화벽 기반의 가상 보안 구역 분할 및 복합 NAT/보안 정책 수립
* **버전/운용도구:** Cisco ASA / GNS3
* **상세 내용:**

  * 가상 방화벽 환경에서의 OSPF/EIGRP 재분배 연동 및 ACL 기반 인/아웃바운드 트래픽 제어 정책 수립
* **나의 역할:** 보안 토폴로지 구상, 가상 컨텍스트 생성 및 MPF 프로토콜 검사 정책 설계, Ping 명령어를 통한 패킷 흐름 검증, 설정 자산 백업 및 기술 보고서 작성

### 5. AWS 클라우드 기반의 고가용성 3-Tier 구축
* **버전/운용도구:** AWS Console
* **상세 내용:** 
  * 가용 영역(AZ 2A, 2C) 기반의 Nginx(Web), Tomcat(WAS), RDS MySQL(DB)로 분리된 이중화 인프라 구성
  * 인터넷 게이트웨이(IGW), NAT 게이트웨이 및 Application Load Balancer(ALB) 연동을 통한 네트워크 이중화와 부하 분산 최적화
  * Route 53 외부 도메인 서브레코드 매핑 및 전 구간 인·아웃바운드 보안 그룹 규칙 정의
* **나의 역할:** 3-Tier 설계 원칙에 따른 클라우드 아키텍처 가이드라인 수립 및 인프라 필수 전제 조건 검토, VPC/서브넷/보안그룹/로드밸런서(ALB) 등의 상세 설정값과 IP 현황을 체계적으로 기술한 '인프라 관리대장' 총괄 작성, Bastion 및 Xshell을 통한 우회 SSH 세션 연결 검증, Nginx-Tomcat 프록시 연동 및 웹-DB 쿼리 테스트 결과 검토 후 최종 기술 문서화 주도
