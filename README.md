## 🚀 Projects & Experience (프로젝트 및 경험)

### 1. AWS 기반 고가용성 웹 서비스 인프라 구축
* **설명:** 단일 장애점(SPOF)을 제거하고 사용량에 따라 서버가 자동 확장되는 인프라를 설계했습니다.
* **주요 작업:** 
  * VPC 및 Public/Private 서브넷 설계를 통한 네트워크 보안 강화
  * ALB(Application Load Balancer)와 Auto Scaling Group 적용
  * NAT Gateway를 통한 Private 서버의 안전한 아웃바운드 통신 구현
* **사용한 기술:** AWS (VPC, EC2, ALB, RDS, IAM)

### 2. Linux 서버 모니터링 환경 구축 및 자동화
* **설명:** 서버의 CPU, 메모리, 디스크 상태를 실시간으로 확인하기 위한 환경을 만들었습니다.
* **주요 작업:** 
  * Prometheus와 Grafana를 연동하여 대시보드 시각화
  * Shell Script를 활용해 디스크 용량 90% 초과 시 경고 알림 자동화
* **사용한 기술:** Linux(Ubuntu), Prometheus, Grafana, Shell Script
